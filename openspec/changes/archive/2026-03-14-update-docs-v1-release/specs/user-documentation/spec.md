# Spec Delta: user-documentation

## ADDED Requirements

### Requirement: v1.0.0 版本更新日志

文档站 SHALL 在更新日志章节提供 v1.0.0 独立页面（英中双语），内容涵盖本版本所有重大新功能与重要修复，且双语页面信息完全对应。

#### Scenario: 用户查阅 v1.0.0 更新内容

- **WHEN** 用户在文档站侧边栏点击 "v1.0.0" 条目
- **THEN** 跳转至对应 changelog 页面，页面展示鼠标支持、波形裁剪、独立音量、签名加密、多语言等新特性说明

#### Scenario: 中文用户查阅更新日志

- **WHEN** 中文用户访问 `/zh/changelog/v1.0.0`
- **THEN** 看到与英文版内容完全对应的中文更新日志，无信息差

---

### Requirement: 首页特性描述反映最新能力

文档站首页（英中双语）的 features 列表 SHALL 包含以下信息，且双语内容完全对应：
1. 鼠标 + 键盘双重支持（含键鼠分离播放模式）
2. 多语言国际化（13+ 语言，含 RTL 支持）
3. 波形可视化裁剪能力
4. 专辑签名保护（防篡改 `.ktalbum`）

#### Scenario: 新用户浏览首页了解产品能力

- **WHEN** 新用户首次访问文档站首页
- **THEN** Feature cards 中可见鼠标支持、多语言、波形裁剪等信息，引导用户进一步阅读使用指南

---

### Requirement: 快速入门页面覆盖新音量功能与语言设置

"快速入门"页（英中双语）SHALL 说明以下新入口，且双语内容完全对应：
- 主页面随机音量变化选项
- 多语言切换（13+ 语言支持）
- 键鼠分离模式的主页面入口

#### Scenario: 用户跟随快速入门配置语言

- **WHEN** 用户阅读快速入门中的"基本设置"章节
- **THEN** 能够找到语言切换说明，知晓软件支持 13+ 语言

---

### Requirement: 安装指南准确描述多平台支持状态

"安装指南"页（英中双语）SHALL 准确表述平台支持情况，说明 macOS 已获得完整支持（含托盘行为、自启动等），双语内容完全对应。

#### Scenario: macOS 用户查阅安装指南

- **WHEN** macOS 用户查阅安装指南"系统要求"章节
- **THEN** 看到 macOS 获完整支持的明确说明，不产生误解

---

### Requirement: 裁剪定义声音指南说明波形裁剪功能

"裁剪定义声音（Step 2）"页（英中双语）SHALL 包含波形可视化裁剪功能的说明，包括以下要点，且双语内容完全对应：
- 波形展示与拖拽选区
- 与时间输入双向同步
- 前端试听播放条（独立于 SDK 预览）
- 音量指针条（±18 dB 可视调节）
- 数字输入裁剪能力保留说明

#### Scenario: 用户使用波形裁剪功能后查阅文档

- **WHEN** 用户在使用波形裁剪时对某功能感到困惑，查阅文档
- **THEN** 文档中有对应功能的清晰说明（选区拖拽、试听、音量条等）

---

### Requirement: 铸造至臻键音指南说明独立音量控制

"铸造至臻键音（Step 3）"页（英中双语）SHALL 包含按下/抬起独立音量控制功能的说明，包括以下要点，且双语内容完全对应：
- 可为按下和抬起声音分别设置独立音量（dB）
- 独立音量与全局音量的叠加关系

#### Scenario: 用户找到独立音量设置说明

- **WHEN** 用户想为按下声音设置独立音量
- **THEN** 在指南第三步章节中找到对应说明

---

### Requirement: 按键联动声效指南涵盖鼠标支持

"按键联动声效（Step 4）"页（英中双语）SHALL 包含鼠标按键支持的说明，包括以下要点，且双语内容完全对应：
- 支持绑定鼠标按键（左键/右键/中键/滚轮）的按下/抬起事件
- 键鼠分离模式说明（可分别指定键盘专辑和鼠标专辑）

#### Scenario: 鼠标用户查阅如何为鼠标按键配置音效

- **WHEN** 用户想为鼠标按键设置音效，查阅按键联动声效页面
- **THEN** 找到鼠标按键绑定的完整说明

---

### Requirement: 键音专辑介绍页说明签名保护体系

"键音专辑介绍"页（英中双语）SHALL 在 `.ktalbum` 格式的相关说明处增加签名保护体系的介绍，且双语内容完全对应：
- 专辑可由作者通过签名进行身份验证
- 签名保护机制可防止专辑内容被篡改

#### Scenario: 用户了解 .ktalbum 安全机制

- **WHEN** 用户在介绍页阅读到 `.ktalbum` 说明
- **THEN** 理解签名保护的用途及其与文件安全性的关系
