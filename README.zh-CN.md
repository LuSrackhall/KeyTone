# KeyTone

## 介绍

KeyTone是一款实用的软件，能够在需要保持安静的环境下实时模拟键盘敲击的声音，从而提升您使用电脑的舒适度。

尽管市面上已经有很多类似的软件，但KeyTone依然致力于为用户提供最佳的体验。

此外，KeyTone是一款遵循GPL协议的开源软件。

## 特性

* 简单到开箱即用。
* 按键的按下和抬起, 拥有独立音效, 完美适配按键长按的场景。
* 用户可以根据自身需求来快速定制按键的声音, 本软件围绕键音专辑设计开发了丰富且便捷的一系列功能。
